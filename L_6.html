This coding task was about replicating and designing different user interfaces using CSS to make them look neat and visually appealing. I worked on four main parts:

  • Google search interface – a search box styled to look clean and easy to use including an input box.
  • Sign-up UI – a form for users to register with input and button element.
  • Booking interface – a design for selecting options and making bookings more organized.

The goal was to practice using CSS to try and replicate the look and feel of each interface while keeping them user-friendly.
